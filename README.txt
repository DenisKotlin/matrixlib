#matrixlib

This library is designed as the first project after learning the syntax of C++, 
to consolidate its basic structures. It allows you to perform the most common 
operations on matrices. All calculations are performed in ordinary fractions, 
thus avoiding rounding errors.

Эта библиотека разработана как первый проект после изучения синтаксиса языка C++, 
для закрепления его базовых конструкций. Она позволяет выполнять наиболее 
распространенные операции над матрицами. Все вычисления выполянются в обыкновенных 
дробях, что позволяет избежать погрешностей округления.
